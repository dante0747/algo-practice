## Excercise #2 - Balanced Expression

Write a solution that checks if the input expression is balanced. The balanced expression is the one in which, for any 
right bracket there's an equivalent left one.

Types of brackets: () {} [] <>

#### Sample:

{(2 + 3) * [3 - 4]} ✔️

({2 + 3) * [3 - 4]} ❌ 

[()<>{}] ✔

[({ >)] ❌
